# Parkinson's Speech Severity ML Project
Machine learning project predicting Parkinson's disease severity from speech features

Course Information

Course: DS 4420 – Machine Learning and Data Mining II
Semester: Spring 2026

By: Rhoda Zerit, Rhea Wadhwa

## Project Overview

Parkinson's disease (PD) affects motor control and speech production. 
Changes in vocal signals may reflect neurological deterioration.

The goal of this project is to use machine learning to predict 
Parkinson's disease severity using speech-based acoustic features.

Instead of classifying whether a patient has Parkinson's disease, 
this project focuses on predicting disease severity using the 
Unified Parkinson's Disease Rating Scale (UPDRS).

## Dataset

The dataset contains biomedical voice measurements from Parkinson's patients
along with clinical severity scores.

Target variable:
motor_UPDRS

## Models

This project will implement multiple machine learning approaches:

1. Neural Network (Python)
2. Bayesian Regression Model (R)

An additional model may be added later for comparison.
